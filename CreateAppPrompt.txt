1. Реализуй в web-приложении my-devices-app, основа которого создана с использованием React JS и Redux Toolkit форму c данными о прикрепленном к пользователю оборудовании.
2. Используй для создания компонентов библиотеку material design в исполнении для React
3. Данные храни в store используя слайсы из Redux Toolkit
4. Приложение  использует данные из firebase, к которой подключается с использованием ключей из env.local. 
5. Вход в эту созданную форму осуществляется по табельному номеру сотрудника вводится в поле ввода, значение его подтом подтягиватеся из коллекции employees из поля tn.
6. После входа в верхней части формы появляются данные из соответствующей tn записи в employees в следующем порядке:
	* tn с надписью на форме "Табельный номер"
	* division
	* position
	* FIO
	* location с надписью на форме "Размещение"
7. Для этого сотрудника из коллекции devices подтягиваются все записи, у которых empID совпадает с идентификатором пользователя. 
8. Связанные с данным пользователем данные из коллекции devices выводятся в карточки на форме, каждая из которых содержит следующие поля в указанном ниже порядке: 
	* nomenclature с надписью на форме "Тип устройства"
	*		
	* serial с надписью на форме "Серийный номер"
	* dateReceipt  с надписью на форме "Дата поступления"
	* status с надписью на форме "Статус"
	* ctc с надписью на форме "КТС"
