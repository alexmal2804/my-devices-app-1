В приложении my-device-app-1 нужно реализоват функионал AI-агента,
который будет отвечать на вопросы пользователя в чате.
Визуальные компоненты нужно реализовывать на React с использованием библиотеки material-ui.
Кнопка вызова AI-агента будет называться "AI-помощник" и будет находиться в модальном окне оборудования в его
правом нижем углу.
После вызова кнопки формируется модальное окно в форме чата с AI-агентом, в котором пользователь может задавать
вопросы и получать ответы от AI-агента.
В это модальное окно передается в составе системного промпта только информация о зарегистрированном сотруднике
и вся информация о оборудовании из данной карточки оборудования.
В чате происходит передача системного промпата модели deepseek-chat с использованием библиотеки langchain, 
через сервис  к OpenAI API c использованием https://aitunnel.ru, ключ aiTonnelKey.
Ключ доступа  к сервису aitunnel.ru берется из env.local.
Агент отвечает на системный промпт кратко рассказывая о себе и сообщая, что может помочь с данным ИТ-оборудованием.
После ответа на системный промпт AI-агент ведет себя как высококвалифицироанный специалист по ИТ-поддержке,
который может ответить на любой вопрос пользователя.
Он отвечает только на вопросы, касающиеся только той единицы оборудования, из карточнки которой было открыто окно чата.
На вопросы не имеющие отношения к данной единице оборудования он вежливо отвечает, что может помочь только по данной
единице, указывая её реквизиты.
Также с использованием langchain происходит сохранение истории диалога в чате, эта история доступна агенту, модель 
отвечает с учетом информации из этой истории. 
После выхода из чата история очищается.
Что в приоритете может сделать агент:
- он может сформировать обращение в поддержку по следующим ситуациям:
    1. На устранение неисправности   
    2. На замену устройства
    3. На открепление оборудования
    4. На возврат оборудовния в подменный фонд
    5. На поиск утерянного оборудования
После формирования обращения агент выводит на экран текст обращения с инлайн-кнопкой "Отправить".
После нажатия кнопки агент отвечает "Обращение отправлено в поддержку. Какие у Вас еще будут вопросы по 
<реквизиты оборудования>?
Если показатель ctc ниже двадцати, то агент сообщает, что имеющееся у вас оборудование подлежит плановой замене.
Если status не равен "исправен", то агент кратко описывает ситуацию исходя из значения поля status.
Температура модели 0,7.
